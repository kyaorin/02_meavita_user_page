# ミアビータ会員専用ページに、１週間の記録と振返り機能追加

## DEMO

  - デプロイしている場合はURLを記入（任意）

## 紹介と使い方

  - 前回作ったミアビータ会員向けのページに１週間の記録と振返り機能を追加

  - １日１問への解答やその日の感情、気付き、モヤモヤ、悩みなどを毎日記録（5/18 13時現在ココ）

  - 月曜〜日曜の記録したものを踏まえて、１週間の振返りができるシートを作成

  - 振返りシートはダウンロードできるようにする
